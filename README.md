# 💻 Laptop Store Management System

A complete Laptop Store Management System developed using JSP, Servlets, JDBC and MySQL with **Role-Based Authentication**.

---

## 🚀 Features

### 👤 User Module
- User Registration
- User Login
- View Available Laptops
- Buy Laptop
- Place Orders
- View Order Success
- Logout

### 👨‍💼 Admin Module
- Admin Login
- Admin Dashboard
- Add Laptop
- Update Laptop
- Delete Laptop
- View All Laptops
- View Customer Orders
- Manage Products

### 🔐 Authentication
- Role-Based Login (Admin & User)
- Session Management
- Secure Logout

---

## 🛠 Technologies Used

- Java

## ⚙️ How to Run

1. Import the project into Eclipse.
2. Configure Apache Tomcat 10.
3. Create the `LaptopStore` database in MySQL.
4. Import the SQL file.
5. Update the database username and password.
6. Run the project.
